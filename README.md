# 考勤管理系统 java1105

## 项目概述

考勤管理系统是基于Spring Boot和Vue的前后端分离架构，适用于各类社团的考勤管理。系统经过实际测试，稳定可靠。

## 技术栈

- **前端技术**：Vue、Element-UI
- **后端技术**：Spring Boot、MyBatis Plus
- **数据库**：MySQL
- **开发语言**：Java

## 功能模块

系统包含以下功能模块：

1. 用户管理
2. 考勤记录
3. 社团活动管理
4. 数据统计与分析

## 系统角色

- 管理员：负责系统管理、用户管理和活动管理等操作。
- 普通用户：进行考勤打卡、活动参与等。

## 运行环境

- **操作系统**：Windows、Linux或macOS
- **数据库**：MySQL 5.7 或更高版本
- **Java版本**：Java 8 或更高版本
- **Node.js**：用于前端项目构建

## 部署步骤

1. 搭建好运行环境，包括数据库和Java环境。
2. 导入MySQL数据库文件。
3. 编译并运行后端服务。
4. 构建并运行前端项目。
